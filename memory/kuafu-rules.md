   - **Why:** 确保工作可恢复，避免再次丢失
   - **Source:** AGENTS.md + 历史数据丢失教训
   - **Added:** 2026-04-29

   - **Source:** AGENTS.md + Nuwa skill 核心原则
   - **Added:** 2026-04-29
